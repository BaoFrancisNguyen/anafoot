Importation des données depuis WhoScored

Présentation

Ce module permet d'importer les données statistiques de football depuis WhoScored.com, un site spécialisé dans les statistiques détaillées de football.

WhoScored propose plus de 200 statistiques brutes pour les joueurs, incluant des catégories comme :

  • Statistiques défensives : tacles, interceptions, dégagements...
  • Statistiques offensives : buts, passes décisives, tirs...
  • Statistiques de passes : passes réussies, passes clés, précision...
  • Statistiques détaillées : duels aériens, fautes, hors-jeu...

Actions disponibles

Importer depuis WhoScored

Importer des équipes, joueurs ou statistiques directement depuis le site WhoScored
Importer depuis un CSV

Charger des données à partir d'un fichier CSV exporté depuis WhoScored ou une autre source

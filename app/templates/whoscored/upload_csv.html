Format des fichiers CSV

Statistiques des joueurs

Le fichier CSV doit contenir les colonnes suivantes :

  • Player : Nom du joueur
  • Team : Nom de l'équipe
  • Position : Poste du joueur
  • Apps : Nombre de matchs joués
  • Goals : Nombre de buts
  • Assists : Nombre de passes décisives
  • Yellow Cards : Nombre de cartons jaunes
  • Red Cards : Nombre de cartons rouges
  • ... et les autres statistiques disponibles

Matchs

Le fichier CSV doit contenir les colonnes suivantes :

  • Date : Date du match (format YYYY-MM-DD)
  • HomeTeam : Nom de l'équipe à domicile
  • AwayTeam : Nom de l'équipe à l'extérieur
  • FTHG : Nombre de buts de l'équipe à domicile
  • FTAG : Nombre de buts de l'équipe à l'extérieur
  • League (optionnel) : Nom de la ligue/compétition
  • Season (optionnel) : Saison (format YYYY/YYYY)
